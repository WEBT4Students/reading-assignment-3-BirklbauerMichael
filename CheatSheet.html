Introduction

World Wide Web

  1. What are the most essential technologies that made the World Wide Web such a success?
    2. The most essential technologies are HTML, URL and HTTP.

  2. How are resources (html pages, images, etc) addressed in the web?
    3. Resources are addressed with URI.

  3. What are the general components of a url?
    4. The URL consists of a hierarchical sequence of five components: sheme, authority, path, query and fragment.

Web Server

  1. What is a web server?
    2. A web server is computer software and underlying hardware that accepts requests via HTTP.

  2. What does a web server?
    3. A user agent, commonly a web browser, initiates communication by making a request for a web page or other resource using HTTP, and the server responds with the content of that resource.

  3. How can content be stored on a web server?
    4. The web space is a hard disk, on which web content is stored and this must be connected to the web server.

  4. How is content in webpages described?
    5. Page content refers to all the information contained in a website. Page content can be displayed as text, links, images, audio, animation or videos among other things.

Web Client

  1. What is a web client?
    2. A web browser is application software for accessing the World Wide Web or a local website.

  2. What does a web client?
    3. When a user requests a web page from a particular website, the web client retrieves its files from a web server and then graphically renders the page on the user's screen.

  3. Web Client and Web Browser is the same?
    4. The web browser acts as an interface between the server and the client and displays a web document to the client.

Hello World

  1. What is the html head element?
    2. The head element is a container for metadata. Metadata is data about the HTML document. Metadata typically define the document title, character set, styles, scripts, and other meta information.

  2. What does !DOCTYPE html mean?
    3. All HTML documents must start with a !DOCTYPE declaration. It is an information to the browser about what document type to expect.

  3. What are the two top most elements within the html element?
    4. The two top most elements within the html element are head and body.

  4. How to emphasize a text and how is it displayed in the browser?
    5. The em tag is used to define emphasized text. The content in the browser is typically displayed in italic.

  5. How to mark up important text and how is it displayed in the browser?
    6. The strong tag is used to define text with strong importance. The content in the browser is typically displayed in bold.

Html and a first look into Css

Html

  1. Components of html elements
    2. The components of html elements are Start-Tag, maybe an Attribute, element content and End-Tag.

  2. Identify void elements and how they are denoted
    3. HTML elements with no content are called void elements. It is closed in its Start-Tag, seen by the slash at the end. The slash can also be omitted. An example is br.

  3. Identify attributes
    4. Attributes provide additional information about elements. All HTML elements can have them. Attributes are always specified in the start tag.

  4. The correct DOCTYPE declaration for html 5
    5. The DOCTYPE declaration for html 5 is < !DOCTYPE html>

  5. Write down syntactically correct statements for the following elements
    • Article
      • The article tag specifies independent, self-contained content.
    • Body
      • The body tag defines the document's body. It contains all the contents of an HTML document, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.
    • line break
      • The br tag inserts a single line break.
    • Headings
      • HTML headings are defined with the h1 to h6 tags. h1 defines the most important heading. h6 defines the least important heading.
    • Section with meta-information
      • The meta tag defines metadata about an HTML document. Metadata is data (information) about data. It define the document title, character set, styles, scripts, and other meta information.
    • Top level element
      • The html element is the root element and it defines the whole HTML document.
    • Image named Team.jpg
      • The HTML img tag is used to embed an image in a web page. Example: < img src="Team.jpg">
    • Paragraph
      • The HTML p element defines a paragraph. A paragraph always starts on a new line, and browsers automatically add some white space (a margin) before and after a paragraph.
    • Section
      • The section tag defines a section in a document.
    • Label appearing in the title bar
      • The title tag defines the title of the document. The title must be text-only, and it is shown in the browser's title bar or in the page's tab.
    • Link to other pages
      • The HTML a tag defines a hyperlink. Most important attribute of the a element is the href attribute, which indicates the link's destination.
    • Comment
      • You can add comments with < !--...-->
    • Head with title and meta elements
      • The head element is a container for metadata (data about data). It is a container for the following elements: title, style, meta, link, script, and base.

Css

  1. Components of a css rule
    2. The components of a css rule are a selector, property and value.

  2. Use combinators properly: direct child, descendant, adjacent sibling, sibling
    3. descendant example: div p { background-color: yellow; }, it selects all p elements inside div elements
    direct child example: div > p { background-color: yellow; }, it selects all p elements that are children of a div element
    adjacent sibling example: div + p { background-color: yellow; }, it selects the first p element that are placed immediately after div elements
    sibling example: div ~ p { background-color: yellow; }, it selects all p elements that are next siblings of div elements

  3. Specify colors by color names and in hexadecimal notation
    4. Tomato in HEX #ff6347, Blue in HEX #0000FF, Cyan in HEX #00FFFF, Green in HEX #008000,....

  4. Declarations for color, background color
    5. The color property is used to set the color of the text. Example: color: blue; You can use many different colors.
    The background-color property specifies the background color of an element. Example: background-color: lightblue; You can use many different colors.

  5. Pseudo classes for the a element
    6. Links can be displayed in different ways, for example: a:link { color: red; } Link stands for unvisited link and you can also use visited(visted link), hover(mouse over link) and active(selected link). Of course you can take a different color for everything.

  6. Declaration for background image
    7. The background-image property specifies an image to use as the background of an element. Example: background-image: url("paper.gif");

  7. Declaration for text alignment (left, right, center, justify)
    8. The text-align property is used to set the horizontal alignment of a text. Example: text-align: center; You can also use right, left and justify.

  8. Declaration for text decoration (overline, underline, line-through)
    9. The text-decoration-line property is used to add a decoration line to text. Example: text-decoration-line: overline; You can also use underline and line-through.

  9. Declaration for text transformation (uppercase, lowercase, capitalize)
    10. The text-transform property is used to specify uppercase and lowercase letters in a text. Example: text-transform: uppercase; You can also use lowercas and capitalize.

  10. Identify the difference between serif and sans-serif fonts
    11. Sans-serif fonts have clean lines and Serif fonts have a small stroke at the edges of each letter.

  11. Declarations for font families
    12. We use the font-family property to specify the font of a text. Example: font-family: Arial; There are many different font-familys.

  12. Declarations for font style normal and italic
    13. The text is shown normally: font-style: normal;
    The text is shown in italics: font-style: italic;

  13. Declarations for font sizes
    14. The font-size property sets the size of the text. Example: font-size: 40px; You can use different font-sizes.

  14. Declarations for font weights
    15. The font-weight property sets how thick or thin characters in text should be displayed. Example: font-weight: bold; You can also take 700 instead of bold and other font-weights.
